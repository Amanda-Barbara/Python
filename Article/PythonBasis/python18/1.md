## python内置函数
## hasattr()函数
```text
hasattr() 函数用于判断对象是否包含对应的属性。
```
```python
#!/usr/bin/python
# -*- coding: UTF-8 -*-
 
class Coordinate:
    x = 10
    y = -5
    z = 0
 
point1 = Coordinate() 
print(hasattr(point1, 'x'))
print(hasattr(point1, 'y'))
print(hasattr(point1, 'z'))
print(hasattr(point1, 'no'))  # 没有该属性
```
输出结果：
```text
True
True
True
False
```

## eval()函数
```text
eval() 函数用来执行一个字符串表达式，并返回表达式的值
```
```text
>>>x = 7
>>> eval( '3 * x' )
21
>>> eval('pow(2,2)')
4
>>> eval('2 + 2')
4
>>> n=81
>>> eval("n + 4")
85
```
